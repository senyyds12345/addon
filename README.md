# addon
Addon is a deb package for quickly creating mcaddon projects, which supports Termux and Linux systems.
# install
- Execute this command to install.
```bash
dpkg -i addon.deb
```
# use
- You must initialize it first before use
```bash
addon ini
```
- This command will check if dependencies are installed. If any dependency is not installed, it will prompt you to enter "y" or "n" â€” enter "y" to download the dependencies, and "n" to exit the installation interface.
- You can use the parameter "--help" to check the command usage tutorial.
